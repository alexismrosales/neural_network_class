--------------------Instrucciones de ejecución
Archivos:
hammingNetwork: Este archivo se encarga de la ejecución del algoritmo de la red de hamming.
inputdata: Su función es leer los archivos de texto disponibles en la carpeta data, solo se debe ejecutar e introducir el número de experimento que se desea realizar.

-----Instrucciones
1. Ejecutar el script "inputdata.m" y elegir la carpeta del experimento deseado con las opciones: 1 | 3 |  4

2. Ejecutar el script "hammingNetwork.m"

-----Importante
Antes de ejecutar el archivo de "hammingNetwork.m" es importante saber que:

En caso que se modifiquen los datos de algún experimento estos deberán volver a leer los datos de los archivos de texto, para ello se debe ejecutar el script "inputdata.m" y seguir las instrucciones, esto generará un archivo donde se guardaran las variables para que el script "hammingNetwork.m" pueda acceder a ello.
